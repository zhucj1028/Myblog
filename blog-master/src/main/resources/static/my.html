关于本站

本网站建于2020年12月12日，主要为了记录个人生活中的所见所闻,前端参考燕十三提供的开源模板进行修改开发，效果还不错。
前 端 ：LayUI等各种插件
后 台 ：基于beyondadmin模板进行修改
后 端 ：SpringBoot + Springsecurity + Mybatis-plus + Mysql
服务器 ：Tomcat + Nginx
本站部署于阿里云ECS服务器。
